# oo-sabor-express

Object-oriented Python project built during the Alura course **"Python: aplicando a Orientação a Objetos"**.

## About

A restaurant management system that explores core OOP concepts in Python, using a food delivery theme (Sabor Express) as the domain context.

## Concepts covered

- Classes and instances
- Instance, class, and static attributes
- Properties and encapsulation (`@property`)
- Class methods and static methods
- `__str__` and `__repr__` special methods
- Optional challenge exercises (Music, Car, Client classes)

## Project structure

```
modelos/
├── restaurante.py   # Main Restaurant class with OOP implementation
└── desafios.py      # Optional Alura challenge exercises
```

## How to run

```bash
python modelos/restaurante.py
```
